# 💻 Dev-Showcase-Portfolio
> **HTML5, CSS3, JavaScript를 활용한 반응형 개인 포트폴리오 웹사이트**

사용자 경험(UX)을 고려한 레이아웃과 반응형 디자인으로 구현된 프론트엔드 포트폴리오 웹사이트입니다.

## 📌 주요 기능 (Key Features)
*   **자기소개 섹션**: 프론트엔드 개발자의 비전과 기술 역량 소개
*   **반응형 레이아웃**: Flexbox와 미디어 쿼리를 활용하여 모바일, 태블릿, PC 등 모든 기기 환경에 최적화
*   **인터랙티브 UI**: 부드러운 스크롤 효과 및 사용자 경험(UX)을 고려한 애니메이션 구현
*   **포트폴리오 쇼케이스**: 진행한 프로젝트들의 결과물을 카드 형태로 배치하여 가독성 증대

## 🛠 기술 스택 (Tech Stack)
*   **Language**: HTML5, CSS3, JavaScript (ES6+)
*   **Concept**: Personal Branding / Responsive Web Design
*   **Tools**: Visual Studio Code, Git
